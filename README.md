# 📝 Simple To-Do List

A minimalistic and interactive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add tasks dynamically
- Remove tasks with a delete button
- Simple and user-friendly UI
- Responsive design 

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
